# Ejercicios-Python
Clase de Procesamiento de Datos

Ejercicios hechos por: Diego Céspedes

A continuación se hace una breve descripción de cada cuaderno de la actividad, hecha después de la ejecución de cada uno de estos cuadernos:

1. En esta primner cuaderno, se ven principios básicos de python, especificamente el manejo de cadenas. Aprendemos como se crean, y que métodos tenemos disponibles, como la concatenación, el manejo de índices o la búsqueda de subcadenas

2. En el segundo cuaderno, aprendemos sobre las tuplas, un tipo de objeto en python que consiste en una serie de elementos que podemos manejar a nuestra discreción. Estos objetos también cuentan con un manejo de indices y más operaciones útiles en la programación.

3. En este tercer cuaderno, exploramos las listas, una de las estructuras de datos más versátiles en python. Vemos cómo modificar elementos, añadir nuevos datos y utilizar métodos específicos para organizar la información de manera dinámica.

4. Aquí nos enfocamos en los conjuntos (sets), objetos ideales para manejar colecciones de elementos únicos. Aprendemos cómo realizar operaciones matemáticas de conjuntos, como uniones e intersecciones, y entendemos por qué su naturaleza no indexada los hace especiales.

5. En el quinto cuaderno, estudiamos los diccionarios, que nos permiten almacenar datos mediante pares de clave-valor. Aprendemos a estructurar información más compleja y a acceder a los valores de forma eficiente, facilitando la organización de datos etiquetados.

6. En este sexto cuaderno, profundizamos en el control mediante el uso de condiciones. Aprendemos cómo hacer que nuestros programas tomen decisiones basadas en comparaciones lógicas, utilizando estructuras como if, elif y else para controlar el flujo de ejecución.

7. Este cuaderno trata sobre los bucles. Analizamos el funcionamiento de for y while, viendo cómo iterar sobre colecciones de datos y controlar la repetición de instrucciones hasta que se cumpla una condición específica.

8. Aquí profundizamos en la modularidad del código a través de las funciones. Aprendemos a definir bloques de código reutilizables, el manejo de parámetros y la importancia de los valores de retorno para escribir scripts más limpios, organizado y eficientes. 

9. En este último cuaderno, nos introducimos en la Programación Orientada a Objetos mediante las clases. Vemos cómo definir objetos propios, integrando atributos y métodos dentro de nuestro código.
